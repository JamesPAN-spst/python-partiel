tuple

  • Immutable, ordonné, hétérogène
  • ⚠ PIÈGE : (1) est un int, (1,) est un tuple à 1 élément
  • Tuple vide : () ou tuple()
  • Opérations : + (concat), * (répétition), in, len(), slicing
  • Hachable (si contenu hachable) → peut être clé de dict ou élément de set
  • zip(A, B) → itérateur de couples (ai, bi)
  • enumerate(L) → itérateur de (indice, valeur)
  • Déballage : a, b, c = (1, 2, 3)
  • ⚠ Immutable = pas de append, sort, del T[i], T[i] = x

frozenset

  • Version immutable de set → hachable
  • Peut être clé de dict ou élément de set
  • Pas de add(), remove(), discard(), pop()
  • Opérateurs &, |, - fonctionnent
  • ⚠ {1, {2,3}} → TypeError ! Utiliser {1, frozenset({2,3})}
  • frozenset & set → renvoie un frozenset
