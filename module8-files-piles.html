File (FIFO)

  • file_vide() -> []
  • ajouter_element(x, F) -> F.append(x)
  • extraire_element(F) -> F.pop(0) O(n) !
  • premier_element(F) -> F[0]
  • Mieux : collections.deque pour O(1)

Pile (LIFO)

  • pile_vide() -> []
  • empiler(P, x) -> P.append(x)
  • depiler(P) -> P.pop() O(1)
  • sommet(P) -> P[-1]

Points cles

  • File = file d'attente (premier arrive, premier servi)
  • Pile = pile d'assiettes (dernier pose, premier retire)
  • La recursivite utilise une pile d'appels systeme
  • Application : verification du parenthesage avec une pile
