Création et propriétés

  • Non ordonné, mutable, éléments uniques et hachables
  • ⚠ Ensemble vide = set(), PAS {} (qui est un dict vide !)
  • set('python') = {'p','y','t','h','o','n'} (6 caractères uniques)
  • Éléments autorisés : int, float, str, tuple, frozenset
  • ⚠ PAS de list ni de set comme élément (non hachables)

Opérateurs ensemblistes

  • A | B = union, A & B = intersection, A - B = différence
  • A <= B = inclusion large, A < B = inclusion stricte
  • ⚠ PRIORITÉ : & > | > -, et TOUS avant in
  • ⚠ PIÈGE MORTEL : 2 in {1,3} & {2,4} = 2 in set() = False

Méthodes

  • add(e) : ajoute e
  • discard(e) : supprime e, silencieux si absent
  • remove(e) : supprime e, KeyError si absent
  • pop() : retire un élément quelconque (KeyError si vide)
  • clear() : vide l’ensemble
