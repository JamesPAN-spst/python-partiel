Vocabulaire

  • Ordre = nombre de sommets, Degre = nb aretes du sommet
  • Complet : toutes les aretes possibles existent
  • Connexe : tout sommet accessible depuis tout autre
  • Cycle : chemin ferme a aretes distinctes

Implementation (dict + set)

  • Sommets = cles, Voisins = valeurs (set)
  • Toutes les operations de base en O(1) en moyenne

Parcours

  • DFS : recursif + marquer les sommets visites
  • BFS : file + marquer les sommets
  • Toujours marquer pour eviter les boucles infinies !

Matrice d'adjacence

  • Non oriente -> matrice symetrique
  • Simple -> diagonale nulle
  • A^k[i][j] = nb chemins de longueur k de i a j

Euler

  • Chemin eulerien = passer par chaque arete exactement une fois
  • Existe ssi graphe connexe + 0 ou 2 sommets de degre impair
  • Somme des degres = 2 * nb aretes
