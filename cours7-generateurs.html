1. Itérables et range

range(10**30) ne crée PAS une liste de 10³⁰ éléments. C'est un itérable paresseux : il génère les valeurs à la demande, sans les stocker en mémoire.

a = range(10**8)     # itérable (prend 48 octets)
sum(a)               # fonctionne → 4999999950000000
b = list(a)          # liste (prend 1 Go !)
c = list(range(10**9))  # Killed (mémoire insuffisante)

3. Fonctions génératrices (yield)

Une fonction avec yield (au lieu de return) crée un générateur. Chaque yield produit une valeur et suspend l'exécution.

def f(a):
    yield a
    yield a * 2
    yield a + 1

list(f(10))     # [10, 20, 11]

# Équivalent avec boucle :
def carres(n):
    for x in range(1, n+1):
        yield x ** 2

list(carres(5))  # [1, 4, 9, 16, 25]
⚠ Épuisement : Un générateur ne peut être consommé qu'une seule fois ! 
g = (x**2 for x in range(5))
list(g)    # [0, 1, 4, 9, 16]
list(g)    # []  ← épuisé !

Épuisement partiel

g = (x**2 for x in range(10))
for i in g:
    if i == 9: break     # consomme x=0,1,2,3 puis s'arrête
list(g)    # [16, 25, 36, 49, 64, 81]  ← le reste

4. Générateurs récursifs

Sous-ensembles

def sous_ensembles(A):
    if len(A) == 0:
        yield set()
    else:
        x = A.pop()
        for B in sous_ensembles(A):
            yield B
            yield B | {x}

list(sous_ensembles({1,2,3}))
# [set(), {1}, {2}, {1,2}, {3}, {1,3}, {2,3}, {1,2,3}]

Permutations

def permutations(T):
    if len(T) == 1:
        yield T
    else:
        for t in permutations(T[1:]):
            for i in range(len(T)):
                yield t[:i] + T[:1] + t[i:]

5. Le module itertools

from itertools import permutations, combinations, product

# Permutations
list(permutations('abc'))
# [('a','b','c'), ('a','c','b'), ('b','a','c'), ...]  → 6

# Arrangements (k éléments parmi n)
list(permutations('abc', 2))
# [('a','b'), ('a','c'), ('b','a'), ('b','c'), ('c','a'), ('c','b')]

# Combinaisons
list(combinations('abc', 2))
# [('a','b'), ('a','c'), ('b','c')]  → 3

# Produit cartésien
list(product((0,1), repeat=4))     # 16 tuples (tous les mots binaires de longueur 4)

# Simuler des boucles imbriquées
for i, j, k in product((1,3,9), range(20), 'abcd'):
    ...  # équivaut à 3 boucles for imbriquées

Applications classiques

# Combien de façons d'écrire 37 = somme de 5 entiers > 0 ?
sum(sum(x)==37 for x in product(range(1,38), repeat=5))  # 58905

# Probabilité de 7 piles en 10 lancers
2**-10 * sum(sum(x)==7 for x in product((0,1), repeat=10))  # 0.171875

# Cryptarithme SIX² = TROIS
for s,i,x,t,r,o in permutations('0123456789', 6):
    if int(s+i+x)**2 == int(t+r+o+i+s):
        print(f'SIX={s+i+x} TROIS={t+r+o+i+s}')
# SIX=169, TROIS=28561

6. any() et all()

any(L)    # True si au moins un élément est True (OU logique)
all(L)    # True si TOUS les éléments sont True (ET logique)

# Exemple : tester si n est premier
def est_premier(n):
    return n >= 2 and not any(n % k == 0 for k in range(2, n))
