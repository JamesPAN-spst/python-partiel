Création

  • {'a': 1, 'b': 2} ou dict(a=1, b=2)
  • ⚠ dict('a'=1) → SyntaxError (guillemets interdits dans syntaxe mot-clé)
  • ⚠ dict('a':1) → SyntaxError (syntaxe accolades uniquement)
  • {} = dict vide (PAS un set vide !)

Accès et modification

  • D[k] : accès, KeyError si k absent
  • D.get(k, v) : renvoie D[k] si k existe, sinon v (jamais d’erreur)
  • D[k] = val : ajoute ou modifie
  • del D[k] : supprime, KeyError si absent
  • Pas de tranches : D[2:4] → TypeError

Parcours

  • D.keys(), D.values(), D.items()
  • k in D équivaut à k in D.keys()
  • for k, v in D.items(): → parcours clé-valeur

Clés hachables

  • Clés doivent être immutables et hachables
  • list → utiliser tuple
  • set → utiliser frozenset
  • ⚠ D[{1,4}] → TypeError unhashable → D[frozenset({1,4})]
