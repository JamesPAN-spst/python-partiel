Structure d'une classe

  • class NomClasse(): pour definir
  • __init__(self, ...) : constructeur
  • __repr__(self) : affichage
  • self = l'objet courant (convention, pas mot reserve)

Surcharge d'operateurs

  • __add__ = +, __sub__ = -, __mul__ = *
  • __pow__ = **, __eq__ = ==, __lt__ = <
  • __getitem__ = []
  • __rmul__ : appele quand l'operande gauche ne gere pas *

Encapsulation

  • self.__attr = attribut interne (cache)
  • Acces direct -> AttributeError
  • Python le renomme en _Classe__attr (name mangling)

Notation objet vs fonction

  • Avant : voisins(G, s) / enfants(A)
  • Avec classe : G.voisins(s) / A.enfants
