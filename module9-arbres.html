Vocabulaire

  • Racine : noeud sans parent (profondeur 0)
  • Feuille : noeud sans enfant
  • Hauteur : profondeur maximale des feuilles
  • Taille : nombre total de noeuds

Implementation Python

  • Noeud = [etiquette, enfant1, enfant2, ...]
  • Feuille = [etiquette]
  • etiquette(N) = N[0]
  • enfants(N) = N[1:]

Parcours

  • DFS (profondeur) : recursif, utilise la pile d'appels
  • BFS (largeur) : utilise une file
  • DFS = pile, BFS = file (a retenir !)

Formules cles

  • Arbre binaire complet de hauteur h : au max 2^(h+1)-1 noeuds
  • Complexite parcours : O(n) pour n noeuds
