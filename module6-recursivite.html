Principes

  • ⚠ Condition d’arrêt OBLIGATOIRE sinon RecursionError
  • Profondeur max par défaut : 1000 → sys.setrecursionlimit(n)
  • PGCD : pgcd(a,b) = a if b==0 else pgcd(b, a%b)

Fibonacci et complexité

  • fib itératif : O(n)
  • ⚠ fib doublement récursif : O(φn) → INUTILISABLE pour n > 40
  • Mémoïsation (dictionnaire cache) : O(n)
  • Exponentiation matricielle : O(log n)

Diviser pour régner

  • Quicksort : moy O(N log N), pire O(N²)
  • Mergesort : toujours O(N log N)
  • Exponentiation rapide : O(log n) multiplications
  • minmax2 : 3n/2 − 2 comparaisons (vs 2(n−1) naïve)

Tours de Hanoï

  • M(n) = 2M(n−1) + 1 → M(n) = 2n − 1
