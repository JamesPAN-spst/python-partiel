1. Principes de la récursivité

Une fonction récursive s'appelle elle-même. Deux ingrédients obligatoires :

def f(n):
    if n == 0:          # 1. CONDITION D'ARRÊT (obligatoire !)
        return 1
    return n * f(n-1)   # 2. APPEL RÉCURSIF (problème plus petit)
⚠ Sans condition d'arrêt : RecursionError: maximum recursion depth exceeded
Profondeur max par défaut : 1000. Modifiable : sys.setrecursionlimit(n)

Mémoïsation — La solution

F = {0: 0, 1: 1}       # cache (variable globale)

def fib_memo(n):
    if n not in F:
        F[n] = fib_memo(n-2) + fib_memo(n-1)
    return F[n]
✓ Principe : Stocker chaque résultat dans un dictionnaire. Si fib(k) est déjà calculé, on le retrouve en O(1) au lieu de recalculer.

3. Diviser pour régner

Principe : découper un problème en sous-problèmes de même nature, les résoudre récursivement, puis combiner les solutions.

Tri rapide (Quicksort)

def tri(L):
    if len(L) == 0:
        return []
    x = L[0]                                    # pivot
    L1 = [a for a in L[1:] if a < x]            # éléments < pivot
    L2 = [a for a in L[1:] if a >= x]           # éléments ≥ pivot
    return tri(L1) + [x] + tri(L2)

Complexité : O(N log N) en moyenne, O(N²) dans le pire cas (pivot toujours min/max).

Tri fusion (Merge sort)

def tri_fusion(L):
    n = len(L)
    if n <= 1:
        return L
    n1 = n // 2
    T1 = tri_fusion(L[:n1])
    T2 = tri_fusion(L[n1:])
    # fusion de T1 et T2 triés
    i1, i2, T = 0, 0, []
    while i1 < len(T1) and i2 < len(T2):
        if T1[i1] < T2[i2]:
            T.append(T1[i1]); i1 += 1
        else:
            T.append(T2[i2]); i2 += 1
    return T + T1[i1:] + T2[i2:]

Complexité : O(N log N) dans TOUS les cas (pire, moyen, meilleur). C'est son avantage sur le quicksort.

ℹ Python : sorted() et list.sort() utilisent Timsort, un hybride tri insertion + tri fusion.

Exponentiation rapide

def puissance(A, n):
    if n == 0: return I          # matrice identité
    if n == 1: return A
    if n % 2 == 1:               # n impair
        return puissance(A, n-1) * A
    B = puissance(A, n // 2)     # n pair
    return B * B

Complexité : O(log n) multiplications. Exemple : A¹⁰² ne nécessite que 9 multiplications.

4. Tours de Hanoï

def hanoi(n, A, B, C):
    if n == 0: return
    hanoi(n-1, A, C, B)    # n-1 disques de A vers C
    print(f'{A} → {B}')    # déplacer le grand disque
    hanoi(n-1, C, B, A)    # n-1 disques de C vers B
