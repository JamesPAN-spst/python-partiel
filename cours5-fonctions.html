1. Fonctions de base

Procédures et return

Une fonction sans return (ou avec return seul) renvoie None :

def afficher(x):
    print(x)        # pas de return

y = afficher(42)    # affiche 42
print(y)            # None
type(y)             # NoneType
✓ Bonne pratique : Tester y is None plutôt que y == None.

Retour multiple

def minmax(L):
    return min(L), max(L)   # retourne un tuple implicite

a, b = minmax([3,1,4])     # a=1, b=4

Fonctions comme objets

Les fonctions sont des objets Python de première classe. On peut les passer en argument :

def appliquer(f, x):
    return f(x)

appliquer(abs, -5)          # 5
appliquer(len, 'hello')     # 5

2. Arguments optionnels

def f(x, y, z=3, L=[], t=True):
    ...
⚠ RÈGLE ABSOLUE : Les arguments optionnels (avec valeur par défaut) doivent TOUJOURS être placés après les arguments obligatoires.

def f(z=3, x): → SyntaxError: non-default argument follows default argument

Appels valides et invalides

Pour def f(x, y, z=3, L=[], t=True) :

AppelValide ?Raison
f(1)✗y manque (obligatoire)
f(1, 3)✓x=1, y=3, reste par défaut
f(y=2, x=1)✓arguments nommés
f(t=False, 1, 2)✗positionnel après nommé
f(1, 1, L=[3], False)✗positionnel après nommé
⚠ PIÈGE : Un argument positionnel ne peut JAMAIS apparaître après un argument nommé.
f(x=1, 2) → SyntaxError

Déballage dictionnaire

d = {'x': 1, 'y': 2, 't': False}
f(**d)    # équivaut à f(x=1, y=2, t=False)
          # z et L gardent leurs valeurs par défaut

3. Lambda — Fonctions anonymes

f = lambda x: x ** 2           # une expression, pas de bloc
g = lambda x, y=1: x // y + x % y   # args optionnels OK

Exemples de calcul :

f = lambda x, y=3, z=True: y - x if z else None

f(3)            # z=True → return 3-3 = 0
f(5, z=False)   # z=False → return None
f(y=1, x=3)     # z=True → return 1-3 = -2

g = lambda x, y=1: x // y + x % y
g(17)           # 17//1 + 17%1 = 17+0 = 17
g(17, 3)        # 17//3 + 17%3 = 5+2 = 7

Fonction d'ordre supérieur

F = lambda f, a, b: a + f(b) * (b - a)
F(lambda t: t**2, 2, 4)   # 2 + (4²)×(4-2) = 2+32 = 34

4. Intégration numérique — Méthode des trapèzes

Approximation de ∫ab f(x) dx avec n subdivisions :

def trapezes(f, a, b, n):
    h = (b - a) / n
    return h * (f(a)/2 + sum(f(a+k*h) for k in range(1,n)) + f(b)/2)
ℹ Formule : Tn = h · (f(a)/2 + f(a+h) + f(a+2h) + ... + f(b-h) + f(b)/2)   avec h = (b−a)/n
