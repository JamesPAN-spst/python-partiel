1. Création d'un dictionnaire

D = {'a': 1, 'b': 2}       # syntaxe accolades
D = dict(a=1, b=2)          # syntaxe mot-clé (clés = identifiants sans guillemets)
D = {}                       # dict VIDE (pas un set !)
⚠ Pièges de syntaxe :
dict('a'=1) → SyntaxError (guillemets interdits dans la syntaxe mot-clé)
dict('a':1) → SyntaxError (la syntaxe clé:valeur est réservée aux accolades)

Clés dupliquées

{'a': 1, 'b': 2, 'a': 3}  # {'a': 3, 'b': 2}
                            # la dernière valeur écrase la première
len({'a':1, 'b':2, 'a':3}) # 2 (pas 3 !)

3. Parcours

D = {'x': 1, 'y': 2, 'z': 3}

for k in D:               # parcours des clés
    print(k)               # x, y, z

for v in D.values():       # parcours des valeurs
    print(v)               # 1, 2, 3

for k, v in D.items():     # parcours clé-valeur (le plus utile)
    print(k, v)            # x 1, y 2, z 3
✓ Raccourci : k in D est équivalent à k in D.keys(). Cela teste les clés, PAS les valeurs.
'x' in D       # True  (clé existante)
1 in D          # False (1 est une VALEUR, pas une clé)
1 in D.values() # True

4. Clés hachables

Les clés doivent être immutables et hachables. Quand un type mutable est nécessaire, on utilise son équivalent immutable :

Type interditRemplacement
listtuple
setfrozenset
D = {}
D[(1, 2)] = 'ok'           # tuple → hachable ✓
D[[1, 2]] = 'ko'           # TypeError: unhashable type 'list' ✗
D[{1, 4}] = 'ko'           # TypeError: unhashable type 'set' ✗
D[frozenset({1,4})] = 'ok' # frozenset → hachable ✓

5. Application — Scrabble (Ex 8)

valeurs = {'A':1,'E':1,'I':1,'L':1,'N':1,'O':1,'R':1,'S':1,'T':1,'U':1,
           'D':2,'G':2,'M':2,'B':3,'C':3,'P':3,'F':4,'H':4,'V':4,
           'J':8,'Q':8,'K':10,'W':10,'X':10,'Y':10,'Z':10}

def score(mot):
    return sum(valeurs[c] for c in mot.upper())

score('JOYEUX')  # J(8)+O(1)+Y(10)+E(1)+U(1)+X(10) = 31
